## 📌 Features

### 📊 1. Visualize Page (Core Feature)
- Enter any **LeetCode question number**
- Fetch problem instantly
- Write your solution
- Click **Visualize / Analyze**

👉 See:
- Step-by-step execution  
- Variable changes  
- Logic flow  
- Error points  

💡 Helps in **deep understanding of code**

---

### 🐞 Debug System
- Analyze your code logic  
- Understand errors deeply  

⚠️ Currently supports:
- ✅ C++  
🚀 Upcoming:
- Java  
- Python  
- JavaScript  

---

### 📈 2. Dashboard / Progress
- LeetCode + Codeforces stats  
- Heatmap (like GitHub)  
- Track consistency  

---

### 🔁 3. Revision Page
- Add questions for revision  
- Select weekdays  
- Get **email reminders**  

Track:
- ✅ Completed  
- ⚠️ Pending  
- ❌ Missed  

---

### 🏆 4. Daily Contest Page
- View contests from:
  - LeetCode  
  - Codeforces  

- Filter contests  
- Add to **Google Calendar**  

---

### ⚙️ 5. Profile Settings
- Add usernames  
- Fetch personal data  
- Separate data for each user

## ⚙️ Installation & Setup

### 1️⃣ Clone the repo

```bash
git clone https://github.com/ayush1234s/visualiseleetcode.git
cd visualiseleetcode
2️⃣ Install dependencies
npm install
3️⃣ Setup environment variables

Create .env file:

VITE_BACKEND_URL=your_backend_url
4️⃣ Run frontend
npm run dev
5️⃣ Run backend
cd backend
npm install
node server.js
📖 How to Use
Go to Settings → Add usernames
Open Visualize Page → Analyze code
Use Revision Page → Schedule revision
Check Contest Page → Stay updated
